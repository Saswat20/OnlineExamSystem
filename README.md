# Online Exam System

## Overview
A web-based Online Exam System built using Java, JSP, Servlets, MySQL, HTML, and CSS.
Students can register, log in, attempt exams, and view results instantly.
This project is designed for interview preparation and follows MVC architecture.

## Features
- Student registration
- Student login (authentication)
- Online exam with MCQs
- Automated evaluation
- Clean UI using CSS
- MVC + DAO structure

## Tech Stack
- Java, JSP, Servlets
- HTML, CSS
- MySQL, JDBC
- Apache Tomcat

## How to Explain in Interview
Explain registration flow, login validation, exam submission, result calculation,
and how MVC separates concerns.
